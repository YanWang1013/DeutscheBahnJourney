# About

<p>A web-app to look at the journey detail information based on the <a href="https://v5.db.transport.rest/">German Train Service's API</a>.</p>

## How to run
- Npm install
```
npm install
```
- build
```
npm run build
```
- run
```
npm run start
```

- Home/Search

- Favorite
